# Donjon FI Runner — DÉPRÉCIÉE

> ⚠️ **Cette extension est dépréciée.**
> Elle est remplacée par **[Donjon FI Compagnon](https://marketplace.visualstudio.com/items?itemName=donjon-fi.donjon-fi-compagnon)**, qui couvre tout ce que faisait Runner et bien plus (analyse, visualisation, aperçu, wiki intégré, etc.).

L'installation de cette extension installe automatiquement Donjon FI Compagnon (via `extensionPack`). Tu peux ensuite désinstaller Runner sans rien perdre.

## Migration

1. Installer **[Donjon FI Compagnon](https://marketplace.visualstudio.com/items?itemName=donjon-fi.donjon-fi-compagnon)** (déjà fait si tu lis ceci, l'extensionPack s'en charge).
2. Désinstaller **Donjon FI Runner**.
3. Le raccourci <kbd>Ctrl</kbd>+<kbd>F5</kbd> et la commande **« Tester le jeu Donjon FI »** continuent de fonctionner — ils sont désormais fournis par le Compagnon.
